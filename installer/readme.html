Windows:

To start the server manually, click on the "eXist Database Startup" icon on the desktop or choose the corresponding entry from the start menu.

You can also install the server as a service, so it will be launched automatically on startup. For this select "Install eXist as Service" from the start menu.

Unix/Other:

Go to the directory where you installed eXist and call bin/startup.sh.

Documentation:

Once the server is running, you can browse to

http://localhost:8080/exist/
